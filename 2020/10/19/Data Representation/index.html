Information representation | Multimedia

Data representation

  • Key terms
    • One's complement
      • each binary digit in a number is reversed to allow both negative and positive numbers to be represented
    • Two's complement
      • each binary digit is reversed and \(1\) is added in right-most positio
    • Sign and magnitude
      • binary number system where left-most bit is used to represent the sign, the remaining bits represent the binary value
    • Binary-coded decimal
      • number systems that uses 4-bits to represents each denary digit
    • ASCII code
      • \(2^{8}\) set of characters
    • Unicode
      • \(2^{16}\) set of characters

Binary Number systems

  • An example \[ \begin{aligned} 128 \quad 64 \quad 32 \quad 16 \quad 8 \quad 4 \quad 2 \quad 1 \\ 2^{7} \quad 2^{6} \quad 2^{5} \quad 2^{4} \quad 2^{3} \quad 2^{2} \quad 2^{1} \quad 2^{0} \end{aligned} \]

Methods to convert from binary to denary and from denary to binary

Binary addition and subtraction | Binary coding

  • One's complement

    • each digit is inverted
    • from \(11111111 \rightarrow 00000000\)

  • Two's complement

    • each digit in the binary number is inverted and a \(1\) is added to the right-most bit.
    • For example \[ \begin{aligned} \quad 1 \, 0 \, 1 \, 0 \, 0 \, 1 \, 0 \, 1 \\ \text{+} \qquad \ \, \ \quad \quad \quad 1 \\ \text{=} \quad 1 \, 0 \, 1 \, 0 \, 0 \, 1 \, 1 \, 0 \end{aligned} \]
    • Formula to calculate the value of a two's complement binary \[ \begin{aligned} B_{2}T_{w}(\bar x) = -x_{w-1} \cdot 2^{w-1} + \sum_{i=0}^{w-2}x_{i}2^{i} \\ for \ (\bar x) \ \text{represents the two's complement number} \end{aligned} \]

  • Sign and magnitude

    • The left-most bit is sign bit \[ \begin{aligned} B_{2}S_{w}(\bar x) = (-1)^{x_{w-1}} \cdot \sum_{i=0}^{w-2}x_{i}2^{i} \\ for \ x \ \text{represent the length} \end{aligned} \]

  • !subtracting a number \(\implies\) adding a negative number
